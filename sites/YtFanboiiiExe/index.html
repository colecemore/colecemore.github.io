What is Dra1n Sites?

Dra1n Sites is a feature within dra1n to create websites for your shortcuts.

How to Create a Dra1n Site

Follow these simple steps to get started building your own Dra1n Site:

  • Step 1: Download unzip, create a new folder, and name it dra1n-site.
  • Step 2: Inside that folder, create a new file called index.html.
  • Step 3: Rename the file to .txt to edit the file
  • Step 4: Add your custom HTML and save the file. You can start with this template:
<!DOCTYPE html>
<html>
  <head>
    <title>My Dra1n Site</title>
  </head>
  <body>
    <h1>Welcome to My Dra1n Site</h1>
  </body>
</html>
  • Step 5: Rename the .txt back to .html and go to dra1n.
  • Step 6: Use the upload feature to upload it and wait for it to be approved.
